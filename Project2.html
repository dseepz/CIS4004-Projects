Project Description

This project creates a fee invoice application for students. This code lets the user go through different options to view student data such as searching/adding a student, deleting a student, printing a fee invoice, and printing a list of students.

Skills Learned

  • Creating functions for each element with public/private static void in Java
  • Using switch cases to navigate through the options listed
  • Using "print" and "println"
